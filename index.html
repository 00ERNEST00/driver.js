Sholo

A light-weight, no-dependency, vanilla JavaScript library to bring certain parts of page in spotlight

Show an Example Learn More

Whats does it do?

Let's you bring any component on the page into the spotlight.

It lets you focus any element of the page while putting an overlay on top of the other elements in page. You can use it to make powerful introduction for different features of your application or for example if you want to highlight some component that needs user attention.

How does it do that?

Ever heard of magic? ...it is not that, it just uses some canvas manipulation to put the focus on some element
